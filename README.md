nodeXpress_basic
================

[![Greenkeeper badge](https://badges.greenkeeper.io/ayxos/node_grunt.svg)](https://greenkeeper.io/)

A basic xpress node web app with grunt (Stylus/Jade) and mongodb

run
```
npm install
```

if you want to view the database, just type
```
show dbs
use gamedb
show collections
db.games.find()
```
